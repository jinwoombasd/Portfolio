# Sound Engineer Portfolio

This is a simple static web portfolio template tailored for live sound engineers (FOH, monitors, system tech, touring).

## Structure

```
Portfolio/
  index.html
  css/
    styles.css
  js/
    main.js
  images/
    clients/
    headshots/
    favicon/
  audio/
    reels/
    tracks/
  docs/
    CREDITS.md
```

## Usage

- Place your profile photo at `images/headshots/profile.jpg`.
- Add MP3 files to `audio/reels/` and `audio/tracks/` (e.g., FOH/monitor reels), then update paths in `index.html`.
- Put cover images in `images/clients/` and duplicate a card to add more works.

## Live-specific docs

- Fill out `docs/STAGE_PLOT.md` with your stage layout.
- Fill out `docs/INPUT_LIST.md` with your channel list and notes.

## Deployment

- Host on GitHub Pages, Netlify, or Vercel.
- Local preview: use VS Code Live Server or a simple HTTP server.

## License/Credits

- Document sources for fonts/icons/images/audio in `docs/CREDITS.md`.
